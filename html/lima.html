Sobre o Curso

A disciplina de Linguagem de Marcação tem como objetivo proporcionar a aquisição de habilidades fundamentais relacionadas ao design, codificação e estilização de páginas web através de linguagem de marcação.

HTML & CSS

Em HTML aprendemos a estruração básica de páginas web, com uso de tags para a formatação de conteúdo, criação de links e inserção de imagens, sendo a base de nossos projetos. Nos estudos de CSS entendemos como estilizar e dar forma as páginas, dando cara ao nosso projeto.

PROJETO INTEGRADOR

LIMA

SOBRE O PROJETO

A pedido do professor Átila, desenvolvemos um site responsivo para apresentar informações sobre as UNIDADES CURRICULARES do primeiro semestre do curso (FPOO, HARE, SOP e LIMA). Cada unidade tem uma página dedicada com detalhes sobre Capacidades Básicas, Capacidades Socioemocionais e Conhecimentos abordados.

CONCLUSÃO

Com nosso conhecimento em HTML, CSS e protótipação no Figma, conseguimos realizar um projeto que não apenas demonstra nossas competências adquiridas em LIMA, mas também fornece uma plataforma informativa e acessível que destaca as habilidades adquiridas em todas as matérias ao longo do curso.
